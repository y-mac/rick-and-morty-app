# Rick and Morty and Products App

This is a React application featuring global state management with Zustand, authentication, and two main feature pages for Rick and Morty characters and products.

## Getting Started

To run the project locally, follow these steps:

1.  **Install Dependencies:**

    ```bash
    npm install
    ```

2.  **Start the Development Server:**

    ```bash
    npm start
    ```

    The application will open in your browser, usually at `http://localhost:3000`.

## Features

*   **Authentication:** Secure login with encrypted state persistence using Zustand and `crypto-js`.
*   **Rick and Morty Characters:** View a paginated list of characters from the Rick and Morty API. Click on a character to see details.
*   **Products:** Browse a paginated list of products from the Fake Store API, displayed in a grid layout.
*   **Glassmorphism UI:** Stylish glassmorphism effects applied to login forms and item cards.
